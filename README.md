# Home Office Settled Status Proxy

## Running the tests

    sbt test it:test

## Running the tests with coverage

    sbt clean coverageOn test it:test coverageReport

## Running the app locally

    sbt run
    
or    

    sm --start HOSS

It should then be listening on port 931021188
